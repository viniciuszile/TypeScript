Dia 2 exemplo 9 - Never

O exemplo a seguir mostra como utilizar o tipo never em TypeScript. O tipo never é usado para funções que nunca retornam, geralmente porque elas lançam um erro ou entram em um loop infinito:

    
      function geradorError(messagem: string, codigo: number): never { // Função que retorna o tipo "never" e lança um erro
          throw { message: messagem, errorCode: codigo }; // A função lança um erro com uma mensagem e um código de erro
      }

      geradorError("deu xabu", 13); // Chama a função geradora de erro com mensagem e código
      console.log(geradorError("deu xabu", 13)); // Não será executado, pois a função irá lançar um erro
