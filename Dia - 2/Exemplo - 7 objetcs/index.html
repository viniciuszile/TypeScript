Dia 2 exemplo 7 - Objetos

O primeiro exemplo mostra como o TypeScript consegue identificar automaticamente os tipos das propriedades de um objeto:

      
        const user = {
            nome: "zile",
            idade: 20,
            sexo: "m"
        }; // O TS automaticamente identifica os tipos das propriedades como string para nome, number para idade e string para sexo

O segundo exemplo define explicitamente os tipos das propriedades de um objeto, caso você queira ser mais específico:

      
        const user2: {
            Nome: string,
            Idade: number,
            Sexo: string
        } = {
            Nome: "zile",
            Idade: 20,
            Sexo: "m"
        }; // Definição explícita dos tipos para as propriedades
