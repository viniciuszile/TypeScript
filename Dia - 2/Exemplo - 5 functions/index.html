Dia 2 - Exemplo 5: Funções

As funções em TypeScript permitem definir o tipo dos parâmetros e o tipo de retorno, proporcionando maior segurança no código.

Exemplo 1: Definindo o Tipo de Retorno

function add (num1: number, num2: number): number {
  return num1 + num2;
}

Neste exemplo, o tipo de retorno é explicitamente definido como number, embora o TypeScript já pudesse inferir isso automaticamente.

Exemplo 2: Tipo de Retorno Inferido

function add2 (num1: number, num2: number) {
  return num1 + num2;
}

Aqui, o tipo de retorno não é explicitamente definido, mas como os parâmetros são números, o retorno será inferido como number.

Chamadas de Função

add(1, 2); retorna 3.

add2(4, 3); retorna 7.
