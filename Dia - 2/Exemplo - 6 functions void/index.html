Dia 2 Exemplo 6 - Functions Void

Exemplo de função que soma dois números:

        
            function add(num1: number, num2: number): number {
                return num1 + num2;
            }

Exemplo de função que imprime o resultado sem retorno:

        
            function result(preco: number) {
                console.log("O resultado é " + preco);
            }

Chamadas das funções:

        
            result(add(5, 5));
