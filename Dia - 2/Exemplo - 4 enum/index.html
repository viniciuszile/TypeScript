Dia 2 - Exemplo 4: Enum

Enums permitem definir conjuntos de constantes com nomes legíveis. Eles facilitam a associação de valores sem precisar usar constantes separadas.

Exemplo de Enum

enum Funcao { adm = 1, dba = 2, driver = 3 }

Exemplo de Objeto Usando Enum

const funcionario = {
  nome: "zile",
  idade: 20,
  funcao: Funcao.dba
}

Neste exemplo, o objeto funcionario usa o enum Funcao para atribuir um valor à propriedade funcao.
